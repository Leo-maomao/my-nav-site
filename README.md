# 毛毛的 AI 智能导航站 🚀

> **"永远相信美好的事即将发生。"**

这是一个为 AI 时代打造的**现代化、响应式、零成本**个人导航与仪表盘系统。它不仅是一个网址收藏夹，更是一个集成了实时资讯、天气、时间管理与访客分析的智能主页。

本项目基于纯 HTML/CSS/JS 构建，**零后端依赖**，极致轻量，完美适配 GitHub Pages 免费部署。

---

## ✨ 核心亮点

### 1. 🤖 智能 AI 资讯聚合 (Smart News Feed)
- **多源聚合**：实时抓取 **机器之心、量子位、36氪、TechCrunch、The Verge** 等国内外顶级科技媒体的 AI 资讯。
- **智能配图**：内置智能算法，当新闻源无图片时，自动根据标题关键词（如 Apple, OpenAI, Robot）匹配高清科技感封面，或生成确定性哈希背景，告别“白图”尴尬。
- **自动轮播**：优雅的卡片式轮播设计，支持触屏滑动。

### 2. 🌦️ 场景化环境感知 (Contextual Dashboard)
- **实时天气**：基于 IP 自动定位，展示当前温度、天气状况及未来 7 天趋势图（支持 Flexbox 响应式布局）。
- **翻页时钟**：复古翻页时钟设计，极简美学，时刻提醒时间的流逝。
- **访客洞察**：集成 51.la 统计，实时展示**当前在线**、**今日访客**、**累计访客**数据，数据真实可信。

### 3. 📱 极致的响应式体验 (Responsive Design)
- **全端适配**：无论是在 4K 显示器、笔记本、iPad 还是手机上，布局都会智能重组。
- **智能缩放**：Banner 采用 `aspect-ratio` 智能比例（大屏 2.5:1，小屏 16:9），拒绝拉伸变形。
- **移动端优化**：侧边栏在移动端自动转换为顶部横向菜单，操作更符合直觉。

### 4. 🛠️ 配置数据与代码分离
- **本地化配置**：内置“配置模式”（仅管理员可见），可在前端直接增删改查工具链接，数据安全存储于本地浏览器 (LocalStorage)。
- **零成本部署**：无需购买服务器或域名，Fork 本仓库即可一键部署至 GitHub Pages。

## 🚀 快速开始 (部署指南)

本项目专为 **GitHub Pages** 优化，3 分钟即可拥有你的专属导航站。

### 第一步：Fork 项目
1. 点击右上角的 **Fork** 按钮，将本项目复制到你的 GitHub 账号下。
2. 建议重命名仓库为 `my-nav` 或 `username.github.io`。

### 第二步：开启 Pages 服务
1. 进入你的仓库，点击 **Settings** (设置)。
2. 在左侧菜单找到 **Pages**。
3. 在 **Build and deployment** -> **Source** 中选择 `Deploy from a branch`。
4. **Branch** 选择 `main`，文件夹选择 `/ (root)`，点击 **Save**。

### 第三步：访问与个性化
等待约 1-2 分钟，GitHub 会生成你的专属链接（如 `https://yourname.github.io/my-nav/`）。
打开链接，点击左下角的 **⚙️ 图标**（需在 URL 后添加 `?admin=1` 参数以开启权限），即可开始添加你常用的 AI 工具。

> **提示**：如需开启管理员模式，请访问 `你的域名/?admin=1`。
